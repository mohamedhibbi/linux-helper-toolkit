# Linux Helper Toolkit

A collection of beginner-friendly Bash scripts and Linux command notes.  
Perfect for practicing Linux, Bash scripting, and GitHub workflow.

## Scripts Included
1. **organize_files.sh** → Organizes files by type.
2. **backup_home.sh** → Creates a backup of your home folder.
3. **system_info.sh** → Shows system information.
4. **network_check.sh** → Checks network status and open ports.

## Notes
- `notes/file_management.md` → File commands
- `notes/networking.md` → Network commands
- `notes/system_info.md` → System commands
